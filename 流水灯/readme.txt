项目名称 ：流水灯实验
项目介绍 ： 实现不同的流水效果的流水灯，并且很容易扩展新的流水效果。
LED.h 定义板子上的LED灯，和对应的函数。通过宏定义还选择想要的LED灯及其函数。
 #define D1 1  选中D1灯
 #define D2 1  选中D2灯
 #define D3 1  选中D3灯
 #define RX 0  不选中RX灯
 #define TX 0  不选中TX灯
还有LED的初始化函数，使用之前应先初始化LED.
void ledInit()；
     
LED.c
  对LED.h中的函数的实现
  
在LED.h、LED.c文件中使用了条件编译，只编译选中的部分。 
 
main.c
定义了两个结构体，一个是用来管理LED灯和其对应的操作函数，另一个是对应的序列的特
效函数（使用ledFunc的LED灯的管理函数实现），  
/*****************************************************************************
*对相应led的操作函数的结构体
*****************************************************************************/
typedef struct ledFunc{
    int led;          //led的序号
    void (*ledOn)();  //开灯函数指针
    void (*ledOff)(); //熄灯函数指针
}LedfunctionArray;

/*****************************************************************************
*led流水特效的函数的结构体
*****************************************************************************/
typedef struct magicFun{
    int num;          //特效的序号
    void (*fun)();    //特效序号对应的函数
}magicLed;
 
 
 
在main函数中简单的调用不同的特效函数，来实现不同的特效闪烁
