看门狗是在软件跑飞的情况下CPU自恢复的一个方式，当软件在选定的时间间隔内不能置位
看门狗定时器（WDT）.WDT就恢复系统。看门狗可用于电噪音，电源故障或静电放电等恶劣
工作环境或高可靠性的环境。如果系统不需要应用到看门狗，则WDT可配置成间隔定时器，
在选定时间间隔内产生中断。WDT的特性如下：4个可选择的时间间隔看门狗定时器模式下产
生中断请求时钟独立于系统时钟，WDT包括一个15位定时/计数器。它的频率由32.768kHz的
晶振决定，用户不能查看计数器的值工作与各个电源模式。让用户了解几种看门狗定时器的
使用。

看门狗的特性如下：
四个可选的定时器间隔
看门狗模式
定时器模式
在定时器模式下产生中断请求

在看门狗模式下，一旦定时器使能，就不可以禁用定时器，也不能改变定时器间隔。
在看门狗模式下，定时器不会产生一个中断请求。
