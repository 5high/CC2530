当用数组进行存储命令映射时，应该将uartCmdRunArray[i].cmd[0]初始化为'\n',不能初始
化为0，应为readString(),如果读取不到数据，str回事储存的会是0，所以防止在执行命令

  /*****************************************************************************
*  函数介绍  ： 初始化串口命令
*            ：
*    参数    ： 无
*   返回值   ： 无
******************************************************************************/
void initUartCmd()
{
  int i;
  for(i = 0; i < ARRAY_SIZE; i++)
  {
    uartCmdRunArray[i].cmd[0] = '\n';
    uartCmdRunArray[i].uartCmdRun_fun = NULL;
  }
}
